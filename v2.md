FadeNote v2 优化说明书

版本：v2 Enhancement
目标：增强体验，不改变核心结构
前提：基于现有 v1 生命周期模型（Active / Archived）

一、P0（必须优先完成）

这些是对产品完成度影响最大的。

1️⃣ 移除 Min，替换为 Delete
目标

简化窗口行为，强化“终点”概念。

调整

当前：

Close

Min

Top 

Pin

调整为：

Close

Top

Pin

Delete

规则

Delete 永久删除

必须二次确认

删除 md 文件 + index.json 节点

不进入回收站

不可恢复

2️⃣ Fading 状态（运行时计算 + 轻微 UI 变化）
目标

让“时间在流动”可被感知。

规则

当满足：

剩余时间 ≤ 1 天
pinned = false
archived = false


触发 Fading 状态。

注意

不新增持久化字段

不修改 index.json

仅运行时计算

UI变化建议

透明度 95% → 85%

文字轻微灰度

纸张边缘更淡

不做动画，不做提醒。

3️⃣ 基础快捷键
最小集合

Ctrl + N → 新建便签

Ctrl + Shift + P → Pin / Unpin

Ctrl + Shift + D → Delete

Ctrl + Shift + A → 打开 Archive

不支持复杂自定义映射。

设置中仅允许修改：

全局唤出快捷键


二、P1（内容表达增强）

这些增强表达能力，但不改变产品本质。

5️⃣ 列表支持（内容层）
支持

有序列表：

1. 内容
2. 内容


无序列表：

- 内容


自动渲染为列表样式。

6️⃣ Todo 支持

语法：

- [ ] 未完成
- [x] 已完成


行为：

点击 checkbox 可切换

切换时更新 md 文件

更新 updatedAt

不支持：

统计数量

自动归档 todo

任务视图

7️⃣ 基础文本样式

支持：

加粗

删除线

下划线

不支持：

字体颜色

字号

对齐方式

图片

表格

保持纸张感。

三、P2（视觉与沉浸体验）Tray 中增加 Setting

优先级较低，但增强质感。

4️⃣ 过期时间可选（设置）
默认

7 天

可选值

1 / 3 / 5 / 7 / 10 / 15 / 20

规则

仅影响未 pinned 的便签

修改后立即影响所有 Active 便签

不影响 Archived


8️⃣ 吸边效果

规则：

距离屏幕边缘 ≤ 20px 自动吸附

仅影响位置

可在设置中关闭

9️⃣ 暗黑模式

要求：

纸张仍保持纸张感

不变成纯黑面板

文字对比清晰

10️⃣ 纸张样式

最多 3 种：

默认纸

泛黄纸

干净白纸

不要做主题商城。

四、非本版本范围（禁止加入）

本版本明确不做：

标签

搜索

文件夹

批量管理

统计

自动永久删除

云同步

图片支持

富文本工具栏

FadeNote 仍然是：

时间驱动的桌面纸片。
