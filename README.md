FadeNote v1 · Copy / Text Checklist

用途：

校验 UI / 文案 / 提示语是否符合 FadeNote v1

不讨论功能，只讨论“说的话对不对”

1. 全局语言规则

 所有文案 仅使用英文

 使用基础英语（A2–B1 水平）

 句子简短、直接

 不使用俚语

 不使用隐喻或文学化表达

 不引用文化、宗教、哲学术语

2. 禁用语义（一旦出现就是 ❌）

文案中 不得出现 以下含义或词汇：

 “Manage / Management”

 “Save forever”

 “Delete permanently”

 “Restore”

 “Recover”

 “Organize”

 “Category / Tag”

 “History”

 “Settings” / “Preferences”

原则：
FadeNote 不解释“怎么管理”，只描述“时间会发生什么”。

3. 时间语义核对（最重要）

每一处涉及便签命运的文案，必须满足：

 使用 fade / expire 表达消失

 不使用 delete 表达自然消失

 Archive 被描述为 时间结果

 没有任何“系统替你决定重要性”的暗示

快速判断题（全部要 Yes）

 用户能否理解「消失 ≠ 删除」？

 文案是否避免“你应该 / 系统会帮你”？

 是否尊重用户选择，而非指导用户？

4. Pin / Unpin 文案核对

 使用 “Pin”

 使用 “Unpin”

 不使用 “Lock / Save / Keep”

 不使用 “Permanent / Forever” 作为按钮文案

 Pin 被表达为 主动选择

 Unpin 被表达为 可逆行为

5. Top / Remove from Top 文案核对

 使用 “Always on Top”

 使用 “Remove from Top”

 不使用 “Cancel Top”

 不使用 “Fixed / Float”

 文案不暗示任何生命周期变化

6. Window Controls（Hover 文案）

逐项核对（必须完全一致）：

 Close → “Close”

 Minimize → “Minimize”

 Top (inactive) → “Always on Top”

 Top (active) → “Remove from Top”

 Pin (inactive) → “Pin”

 Pin (active) → “Unpin”

7. Tray Menu 文案

 “New Note”

 “Show Notes”

 “Archive”

 “Quit”

核对要点：

 使用动词 “Show”，而不是 “Restore”

 Archive 不加任何修饰词

 Quit 是唯一退出相关文案

8. Show Notes 文案语义核对

 文案不暗示“找回丢失内容”

 文案不暗示“管理隐藏内容”

 Show Notes 被理解为：

make visible again

 不使用 “hidden list / background notes”

9. Archive 页面文案核对

 Archive 被描述为过去

 Archive 文案中不出现“恢复”

 不暗示用户可以“再使用”

 不出现“storage / backup”语义

空状态文案检查

 不指责用户（no notes yet 是 OK）

 不引导用户操作

 只描述时间结果

10. First Note 文案核对（逐句）

必须 完全一致，不允许改写：

What you write here won’t stay forever.

This note will fade after some time,
unless you choose to pin it.

FadeNote doesn’t keep everything —
only what matters right now.


核对点：

 明确 “won’t stay forever”

 明确 “fade”

 明确 Pin 是唯一例外

 不解释规则细节

 不出现品牌自夸

11. 提示 / Toast / 弹窗文案（如有）

 尽量没有

 若存在，仅描述事实

 不说“success / completed”

 不进行情绪化反馈

12. 最终一致性审查（必做）

对每一处文案，回答下面三个问题：

 这句话是在解释时间，而不是教人用软件？

 这句话有没有偷偷替用户做判断？

 如果只读这句话，是否仍符合 FadeNote 的气质？

只要有一题是 No，这条文案就必须改。

✅ 文案 Definition of Done

所有 checklist 项为 ✅

文案不解释、不教学、不劝导

文案只做一件事：
如实描述“存在与消失”
